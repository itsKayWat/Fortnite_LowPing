A specialized tool designed to optimize network performance for Fortnite players, offering both light and full versions for different needs.

Available Versions

Light Version (Recommended)

  • Safe to use during active gameplay
  • Automatic optimization reversal
  • Minimal system impact
  • Quick and efficient

Full Version

  • Aggressive optimization
  • Extended feature set
  • Deep system modifications
  • Maximum performance impact

System Requirements

  • Windows 10 or 11
  • Python 3.7 or higher
  • Administrator access
  • Active internet connection

Installation Steps

  1. Download and install Python from python.org
  2. Clone or download this repository
  3. Run requirements.py as administrator
  4. Choose your preferred version

Usage Guide

Light Version:

  1. Run as administrator
  2. Select optimization option
  3. Play Fortnite
  4. Optimizations auto-reverse on game close

Full Version:

  1. Run before starting Fortnite
  2. Select optimization level
  3. Apply additional options if needed
  4. Launch Fortnite
  5. Manually reverse when finished
